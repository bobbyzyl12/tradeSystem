# TradeSystem
  A trade system including the trader side and the broker side.   
  You can see them in the 'trader' branch and the 'broker' branch.


### Environment
- Server: Tomcat 7.0  
- Language(mainly): Java & html & js  
- frame: SSM(Spring & Springmvc & maven)  
- database: mysql  
- Middleware：JMS
